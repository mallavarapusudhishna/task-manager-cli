# Task Manager CLI

A simple command-line task manager built with Python.

## Features

- Add tasks
- List tasks
- Complete tasks
- Delete tasks
- JSON-based storage

## Usage

Add task:
```bash
python app.py add "Learn Python"
```

List tasks:
```bash
python app.py list
```

Complete task:
```bash
python app.py complete 1
```

Delete task:
```bash
python app.py delete 1
```
